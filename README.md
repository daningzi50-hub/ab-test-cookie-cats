# Cookie Cats A/B 测试：强制关卡该不该移到第40关

## 背景
Cookie Cats 拟将首个强制关卡从第30关（对照组）移到第40关（实验组），用A/B实验（9万玩家）评估对次日/7日留存的影响，决策是否上线。

## 方法
数据质检（无缺失/重复，剔除轮次离群点仅用于展示）→ 卡方检验SRM → 描述性统计与可视化 → 两比例z检验（比例型变量，非均值比较）给出p值与95%CI → bootstrap交叉验证。

## 核心结论
- SRM：p=0.0086，两组49.56%/50.44%，偏差仅0.88pp，判定轻微、不影响分析。
- 次日留存：44.82% vs 44.23%，差0.59pp，p=0.074，**不显著**。
- 7日留存：19.02% vs 18.20%，差0.82pp，p=0.0016，**显著**，bootstrap一致（gate_30更优概率99.92%）。
- 按月增10万用户估算，移至40关每月约减少820名7日留存用户。

## 决策建议
**不建议上线，维持第30关。** 次日留存差异不显著；7日留存显著下降且影响用户规模可观，统计与业务显著并存。若仍想推进，建议改更温和方案（如35关）或延长实验期重评。
